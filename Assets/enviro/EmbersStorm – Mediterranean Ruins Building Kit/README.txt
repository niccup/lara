OVERVIEW
--------
A modular Mediterranean and North African inspired ruins building kit
featuring authentic Tunisian and Middle Eastern architecture. Build
destroyed villages, ancient medinas, desert outposts, and war-torn
environments with AAA quality photogrammetry textures optimized for
real-time performance.

Inspired by real architecture from Jerba, Tunisia — domed shrines
(baraka), flat rooftops with skylights, ornamental tile floors, and
weathered plaster walls. Perfect for open world, FPS, RPG, and
survival games.


PACK CONTENTS
-------------
- 90+ Game-Ready Prefabs
- 14 Materials (tileable, swappable)
- 1K Photogrammetry Textures (Albedo, Normal Map, AO)
- Fully optimized low poly meshes
- Clean UV mapping on every asset
- Demo scene included

PERFORMANCE & OPTIMIZATION
---------------------------
  - All prefabs are low poly and optimized for mobile and PC
  - Meshes use efficient triangle counts with no wasted geometry

  - Supports URP and Built-in Render Pipeline

  - No scripts included — pure art assets, drag and drop ready


HOW TO USE
----------
  1. Import the package into your Unity project
  2. Open the Demo Scene to see all prefabs assembled
  3. Drag prefabs from the Prefabs folder into your scene
  4. Snap pieces together using Unity's grid snapping (hold CTRL)
  5. Swap materials freely — all walls share the same UV layout
     so any material works on any wall prefab
